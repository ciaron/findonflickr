findonflickr by ciaron

Nautilus extension for finding the Flickr page for an image previously downloaded from Flickr.

Dependencies

gnome python-nautilus

Install

Copy findonflickr.py to your Nautilus extensions folder (usually .nautilus/python-extensions) and restart Nautilus (easiest way: logout and log back in).
